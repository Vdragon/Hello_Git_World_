## 以 `help` Git 命令查看指定 Git 命令的使用手冊
您可以以下列命令查閱各個 Git 命令的使用手冊：
`````
$ git help 〈Git 命令名〉
（等價於 $ man git-〈Git 命令名〉）
`````

比方說 `$ git help clone` 或是 `$ man git-clone` 可以查閱 `clone` Git 命令的使用手冊，其中會有這個命令支援哪些參數以及參數的用途等說明。
